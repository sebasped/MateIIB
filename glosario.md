# Glosario

:::{glossary}

Jupyter Notebook
: Entorno de programación que combina texto y código. En Python se llaman Jupyter Notebooks. Se abrevian NB (de NoteBook) y los archivos son `.ipynb` (`i`nteractive `py`thon `n`ote`b`ook).

Python
: Lenguaje de programación.


:::
