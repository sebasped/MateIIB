# Bibliografía, material y recursos

## Para Python y Programación

- [Videos uso básico Python](/material/python_uso_basico.md).
- [Python Programming And Numerical Methods: A Guide For Engineers And Scientists](https://pythonnumericalmethods.studentorg.berkeley.edu/notebooks/Index.html):
    - [PART I INTRODUCTION TO PYTHON PROGRAMMING](https://pythonnumericalmethods.studentorg.berkeley.edu/notebooks/Index.html#part-i-introduction-to-python-programming): referencia general para aprender a programar en Python.
- [Python for Biologists](https://www.pythonforbiologists.org/): colección de videos, código y ejercicios, para aprender Python a través de ejemplos de genómica.
- {cite}`10.1371/journal.pcbi.1004867`: aprender Python con mirada Biocientífica. Se puede descargar.
- [Think Python, Allen Downey](https://allendowney.github.io/ThinkPython/): excelente referencia para aprender Python. Cada capítulo incluye Jupyter NBs con ejemplos y ejercicios.
- [Más libros de Python de Allen Downey](https://greenteapress.com/wp/): más libros de este autor. Todos se pueden descargar o leer online, e incluyen tantos explicaciones como Jupyters NBs.
- [Tutorial de la documentación oficial de Python](https://docs.python.org/3/tutorial/index.html).

## Para la materia propiamente dicha
- [Apunte teórico de la materia](/material/Mate_II_Biologia_apunte.pdf).
- [Python Programming And Numerical Methods: A Guide For Engineers And Scientists](https://pythonnumericalmethods.studentorg.berkeley.edu/notebooks/Index.html):
    - [PART II INTRODUCTION TO NUMERICAL METHODS](https://pythonnumericalmethods.studentorg.berkeley.edu/notebooks/Index.html#part-ii-introduction-to-numerical-methods):  sigue bastante el espíritu de la materia. Mezcla explicaciones con código Python.

## Recursos varios de interés
- [Biopython](https://biopython.org/): herramientas en Python para Biología Computacional.
- [Teach Yourself Computer Science](https://teachyourselfcs.com/): recursos de programación y computación propiamente dichos, para aprender literalmente casi todo.
